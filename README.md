# Let's Organized

A native macOS app that brings every project on your Mac into one place —
git status, tasks, a visual planning board, kanban, deadlines & reminders,
and one-click launch into your favorite editor.

### Download
**→ [Download the latest version](https://github.com/rotem221/lets-organized-app/releases/latest)**

Universal (Apple Silicon + Intel) · macOS 14 or later.
Open the DMG and drag **Let's Organized** to your Applications folder.
