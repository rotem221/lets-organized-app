Privacy Policy

Last updated: July 2026

Let's Organized is a local-first desktop app for macOS and Windows. Your privacy is respected by design: the app is built so that your data stays on your own computer.

What the app stores

Everything you create — your list of projects, tasks, notes, boards, tags, categories and settings — is saved in a single file on your own device (library.json, in your user Application Support / AppData folder). This data is never sent to us, and we have no server that receives it.

If you enable iCloud Drive sync (macOS), that file is synced by Apple to your own iCloud account under Apple's terms — not to us.

Network requests the app makes

  • Update checks. When you check for updates (or on launch, if enabled), the app requests the latest release information from GitHub. This is a normal web request to GitHub and is handled under GitHub's privacy policy.
  • Git & editors. The app reads your local project folders and can run your locally-installed tools (git, editors). These run on your machine; nothing is sent to us.

The app contains no analytics, no advertising, and no third-party tracking.

This website

This site is a static page hosted on GitHub Pages. Like any website, the hosting provider (GitHub) may process technical information such as your IP address and browser type in its server logs to deliver the page and for security. We do not add analytics or tracking cookies to this site.

Data we collect

We do not collect, store, sell, or share any personal data. There is no account and no sign-in.

Children

The app is not directed at children under 13 and collects no data from anyone.

Changes

We may update this policy; material changes will be reflected on this page with a new date.
